
Simple Password Strength Analyser:

• Project Overview -->
The Simple Password Strength Analyser is a Python-based console application designed to help users evaluate the strength of their passwords. It analyzes the password based on length, presence of digits, uppercase letters, and lowercase letters. The program also provides useful suggestions to improve weak or medium passwords and allows multiple attempts until the user decides to exit.
This project demonstrates fundamental Python programming concepts such as loops, conditionals, string methods, counters, and user input handling.


• Features-->

1.) Confirms password by matching input and confirmation Counts:
Digits,
Uppercase letters,
Lowercase letters,
Checks password length,

2.) Classifies password strength as:
WEAK,
MEDIUM,
STRONG,

3.) Provides improvement suggestions.

4.) Allows multiple attempts.

5.) Displays total number of attempts.

• How to Run the Program -->
Python should be installed on your system.

Save the code in a file named:
password_strength_checker.py

Open Command Prompt / Terminal

Navigate to the file location

Run the command:
python password_strength_checker.py
